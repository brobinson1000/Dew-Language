# Dew-Language

# High-Level English-Like Programming Language

A high-level programming language inspired by **COBOL** and **OCaml**, implemented in **C++** and **C**.  
This project explores **compiler and interpreter design, object-oriented programming, and memory management**.

---

## Features

- **English-like syntax** for readable, high-level code.  
- **Automatic memory management**: Heap-allocated objects are cleaned up at program termination.  
- **Optional manual memory management**: Users can explicitly free resources when needed.  
- **Object-oriented and functional paradigms**: Combines classes, objects, and OCaml-inspired functional programming concepts.  
- **Extensible runtime/interpreter**: Supports experimentation with new language features and execution models.  

---

## Key Learning Outcomes

- **Interpreter and compiler design**: Implemented parsing, abstract syntax trees, and runtime execution.  
- **Object-oriented programming**: Designed class structures, encapsulation, and object interactions.  
- **Memory management**: Gained hands-on experience with automatic and manual heap allocation.  
- **Programming language design**: Explored functional and object-oriented paradigms, language semantics, and runtime behavior.
